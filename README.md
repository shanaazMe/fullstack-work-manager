# Full-Stack Task & Productivity Manager

A streamlined, lightweight asynchronous web application featuring a Node.js backend architecture to handle workflow operations and data routing.

## Tech Stack
* **Backend:** Node.js, Express.js
* **Frontend:** HTML5, CSS3 (Responsive Design)
* **API Architecture:** RESTful APIs (GET, POST methodologies)

## Features
* **Asynchronous Data Fetching:** Utilizes JavaScript Async/Await fetching mechanisms to communicate with the server without page reloads.
* **REST API Endpoints:** Implements clean routing for data management.
* **Responsive UI:** Clean, minimalist dashboard designed with modern CSS layout variables for optimal cross-device viewing.

## How to Run Locally
1. Clone or download this repository.
2. Ensure you have Node.js installed.
3. Open your terminal in the project folder and run:
   ```bash
   npm init -y
   npm install express
   node server.js
